HelpDen -Live Support Chat. This plugin integrates HelpDen with WordPress - an application for online live customer support.

== Description ==

HelpDen is a simple and scalable Live Support system that can be added to your website in minutes.

Requires a FREE account at www.helpden.com

== Installation ==

1. Upload `wordpress-helpden` directory to Wordpress plugins directory (`/wp-content/plugins/`)
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Click the 'HelpDen' menu on the left.
4. Follow the instructions.
